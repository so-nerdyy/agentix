# Phase 15 - Conversational Loop

## Purpose

Make the agent usable for ongoing back-and-forth work instead of only one-shot tasks.

## Scope

- Conversation state.
- Task creation from natural language.
- Review and follow-up loops.

## Implementation

- Session-based chat history.
- Streaming replies.
- Task/result correlation.

## Acceptance Criteria

- Users can continue conversations naturally across multiple turns.
- Responses are attached to the right task and session.
