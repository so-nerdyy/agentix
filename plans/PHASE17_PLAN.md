# Phase 17 - Extensibility and Marketplace

## Purpose

Allow third-party extensions without breaking core behavior.

## Scope

- Extension discovery.
- Marketplace metadata.
- Install/enable/disable flows.

## Implementation

- Extension lifecycle and trust model.
- Compatibility checks.
- CLI and shell surfaces for management.

## Acceptance Criteria

- Extensions are manageable as first-class product features.
- Core execution remains Agentix-owned.
