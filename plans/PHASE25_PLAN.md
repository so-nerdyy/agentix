# Phase 25 - Final Ship Gate

## Purpose

Define the final completeness bar for an Agentix release.

## Scope

- Terminal shell.
- Dashboard.
- Channels.
- Memory.
- Approvals.
- Setup/update/doctor.
- Packaging and install flow.

## Implementation

- Release packaging and install checks.
- Clean-machine smoke test.
- Launch readiness and parity review.

## Acceptance Criteria

- `agentix` works after a global install.
- The shell, backend, and dashboard agree on task state.
- The product is shippable without hidden manual steps.
